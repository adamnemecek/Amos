DSMI

MIDI over WiFi

Mac OS X Setup:

  • Download and install the "DSMI Server for OSX" from http://code.google.com/p/dsmi/downloads/list
  • Start the DSMI server application on your Mac
  • Start Amos on your iPad
  • Start your MIDI capable desktop app (Pro Tools, Logic, Live, etc.)
  • In that app, select DSMIDIWiFi-out from the list of available MIDI inputs
  • You should now be receiving MIDI notes from Amos to your desktop app. Email support@anythinghonest.com if you need further help.

Windows Setup:

  • Download and install the "DSMI Server for Windows" from http://code.google.com/p/dsmi/downloads/list
  • You need a Windows MIDI loopback driver installed. MIDI YOKE NT 1.75 is free and recommended
  • Start the DSMI Server and MIDI YOKE on your Windows machine
  • Start Amos on your iPad
  • In DSMI Server, select In From MIDI Yoke: 1 as Input and Out to MIDI Yoke: 2 as output
  • Start your MIDI capable desktop app (Pro Tools, Sonar, Live, etc.)
  • In that app, select MIDI YOKE :2 from the list of available MIDI inputs
  • You should now be receiving MIDI notes from Amos to your desktop app. Email support@anythinghonest.com if you need further help.
